## What is Personal Finance API?

Personal Finance API is a RESTful backend service that lets you track income and expenses with ease. Built on **Node.js** and **Express**, it provides a clean, layered architecture with full CRUD for transactions, category management, and powerful financial statistics.

## Key features

## API overview

The API exposes three resource groups under `/api`:

| Resource | Base path | Description |
|---|---|---|
| Transactions | `/api/transactions` | Income and expense records |
| Statistics | `/api/stats` | Aggregated financial reports |
| Categories | `/api/categories` | Transaction category management |

## Built with

- **Node.js** — JavaScript runtime
- **Express 5** — Web framework
- **Zod** — Schema validation
- **dotenv** — Environment configuration
- **CORS** — Cross-origin resource sharing
