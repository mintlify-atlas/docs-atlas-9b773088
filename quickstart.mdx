---
title: "Quickstart"
description: "Get the Personal Finance API running locally and create your first transaction in minutes."
---

<Steps>
  <Step title="Clone the repository">
    Clone the project to your local machine:

    ```bash
    git clone <your-repo-url>
    cd expense-tracker
    ```
  </Step>

  <Step title="Install dependencies">
    Install the required Node.js packages:

    ```bash
    npm install
    ```
  </Step>

  <Step title="Create the .env file">
    Create a `.env` file in the project root with the following variables:

    ```bash .env
    PORT=3000
    CORS_ORIGIN=http://localhost:3000
    ```

    <Note>
      `CORS_ORIGIN` is required — the server will throw an error on startup if it is missing. See [Authentication](/authentication) for details on configuring multiple origins.
    </Note>
  </Step>

  <Step title="Start the development server">
    Start the server in watch mode so it restarts automatically when you edit files:

    <CodeGroup>

    ```bash npm run dev
    npm run dev
    ```

    ```bash node --watch
    node --watch app.js
    ```

    </CodeGroup>

    You should see the following output:

    ```
    Servidor ejecutándose en puerto 3000
    ```

    The API is now available at `http://localhost:3000`.
  </Step>

  <Step title="Create your first transaction">
    Send a POST request to create an expense transaction:

    ```bash
    curl -X POST http://localhost:3000/api/transactions \
      -H "Content-Type: application/json" \
      -d '{
        "type": "expense",
        "amount": 45,
        "category": "food",
        "description": "Grocery shopping",
        "date": "2026-03-15"
      }'
    ```

    A successful request returns the created transaction:

    ```json
    {
      "id": "a3f1c2d4-8b0e-4e2a-9f6d-1c2b3a4d5e6f",
      "type": "expense",
      "amount": 45,
      "category": "food",
      "description": "Grocery shopping",
      "date": "2026-03-15"
    }
    ```
  </Step>
</Steps>
