Personal Finance API is a Node.js and Express backend that gives you full control over your financial data. Use it to record income and expense transactions, organize them by category, and query aggregated statistics — all through a clean REST interface.

## API overview

The API exposes three resource groups under `/api`:

| Resource | Base path | Description |
|---|---|---|
| Transactions | `/api/transactions` | Create, read, update, and delete income and expense records |
| Statistics | `/api/stats` | Aggregated reports: balance, category breakdown, monthly comparison |
| Categories | `/api/categories` | Manage the categories used to classify transactions |

## Built-in categories

The following categories are available out of the box:

`food` `transport` `entertainment` `utilities` `health` `shopping` `salary` `other`
